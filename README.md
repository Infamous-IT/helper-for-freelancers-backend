# Helper For Freelancers
Helper For Freelancers is a web application that will help every freelancer to be more productive and not worry about forgetting things. Thanks to this application, you will be able to keep track of your orders, create client lists, and make notes (remarks) on orders. 

## Getting started

To get started you will need:

    - Tomcat 9.0.62

    - Java 11

Before you start the project you need to do the following:

    - mvn clean - for remove obsolete files

    - mvn install - for install all dependencies

    - mvn clean install - for remove obsolete files and install new dependencies
